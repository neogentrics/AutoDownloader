Latest Stable (v1.8.1) Features

  • Content Verification: Checks downloaded files against official TMDB episode counts to warn about missing episodes.
  • Settings System: A new Preferences window (Ctrl+P) to manage API keys (TMDB/Gemini) and default settings.

üö® Known Issues in v1.8.1

  • App Crash: Application crashes if a metadata lookup fails (e.g., from a Season 2 URL).
  • URL Parsing: Fails to parse show names from Season 2+ URLs or YouTube playlist titles, causing metadata failure.
  • Playlist Item Limit: Downloads from some sites (e.g., Tubi) are incorrectly limited to 20 episodes.

Future v2.0 Roadmap

The next major version will be a complete overhaul focusing on reliability and user experience.

  • **UI Overhaul:** Implement dynamic download progress bars (Percentage, Speed, ETA) and move the verbose output to an optional, toggleable **Developer Log**.
  • **Scraper Engine Refactor:** Fix all URL/playlist parsing bugs and implement smarter page text scraping.
  • **Multi-Scraper Integration:** Add TVDB as a user-selectable pop-up option and integrate external subtitle databases.
  • **Branding:** Design a new application icon and official name.
