v1.8.1 Release Summary (Critical Bugfix)

Fix: The primary bug where direct URLs defaulted to 'How It's Made' for metadata lookup has been resolved by implementing URL path parsing.

  • **Critical Fix:** Direct URL parsing now dynamically extracts the correct show title (e.g., 'Love Thy Neighbor') for TMDB lookup and naming.
  • **Stability Fix:** Finalized compilation by resolving stubborn nullable type conflicts (`int?`) in `MetadataService.cs`.

v1.8.0 Feature Implementation

The first step toward Content Verification and User Configuration.

  • **Content Verification:** Added structure to count downloaded files and check against official TMDB episode counts.
  • **Settings System:** Implemented Preferences UI (Ctrl+P) and backend service for API key management.
  • **Validation:** Added launch-time pop-up warnings for missing required API keys.
